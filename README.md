## Descrição

O objetivo deste projeto era a criação e manutenção de uma lista de tarefas associada a um usuário distinguido por nome de usuário e senha. 
Foi usado Java Maven e Spring Boot para o desenvolvimento deste, além de H2 Database, e por fim implantado temporariamente no Render.
O programa tem por funções a criação de tarefas, sua atualização e listagem de todas as tarefas criadas pelo usuário. 
O uso começa pela criação do usuário, constando os atributos Nome ("name"), Nome de Usuário ("username) e senha ("password"), recebendo uma Id gerada automaticamente. O usuário passa uma validação das informações e logo pode criar a primeira tarefa constando os campos Título ("title"), Descrição ("description"), Prioridade ("priority"), Data de Início ("startAt") e Data de Término ("finishAt"), a tarefa recebendo uma Id gerada e tendo a data de criação da tarefa marcada automaticamente. Os campos escritos pelo usuário podem ser alterados somente pelo mesmo, e podem ser listados também.
